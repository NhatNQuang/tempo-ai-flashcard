The base surface. Everything that isn't full-bleed sits on a Card. Use `interactive` for clickable cards (study spaces, notes, groups) to get the hover lift.

```jsx
<Card padding="md">…</Card>
<Card interactive onClick={open}>…</Card>
<Card glow>Featured</Card>
```

Padding `none|sm|md|lg`. Compose stats, progress rings, badges, and avatar groups inside.
