User avatar — image with auto initials + deterministic color fallback.

```jsx
<Avatar name="Minh Nguyen" src={url} size="lg" ring />
<Avatar name="Emily Johnson" verified />
```

`ring` marks the current user, `status="online"` adds a presence dot, `verified` adds the blue check. Sizes xs/sm/md/lg/xl → 22–56px.
