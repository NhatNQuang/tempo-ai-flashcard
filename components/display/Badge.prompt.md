Status / level pill. Note levels and visibility tags use `uppercase`; counts and labels use mixed case.

```jsx
<Badge tone="violet" uppercase>Detailed</Badge>
<Badge tone="blue" uppercase>Normal</Badge>
<Badge tone="green" uppercase>Basic</Badge>
<Badge tone="neutral" uppercase>Public</Badge>
<Badge tone="amber" variant="solid">Pro</Badge>
<Badge tone="green" dot>Active now</Badge>
```

Tones: violet/blue/green/amber/red/neutral. Variants: `soft` (default tinted), `solid`, `outline`. Convention: DETAILED‚Üíviolet, NORMAL‚Üíblue, BASIC‚Üígreen.
